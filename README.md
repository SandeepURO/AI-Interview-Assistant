# AI Interview Preparation Assistant

An AI-powered interview preparation platform built using Flask, HTML, CSS, and JavaScript. The application helps students and job seekers prepare for technical interviews through role-based questions, answer explanations, progress tracking, AI feedback, and PDF export functionality.

---

## Features

* Role-based interview question generation
* Multiple career tracks

  * Machine Learning Intern
  * Data Science Intern
  * Python Developer
  * Software Engineering Intern
  * Frontend Developer
  * DevOps Intern
* Difficulty levels for questions
* Show/Hide answer functionality
* Interview score tracking
* Progress bar visualization
* Performance summary dashboard
* AI feedback system
* PDF export support
* Flask REST API backend

## How to Run

### Clone Repository

```bash
git clone https://github.com/SandeepURO/AI-Interview-Assistant.git
cd AI-Interview-Assistant
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install flask flask-cors
```

### Run Backend

```bash
cd backend
python app.py
```

### Open Frontend

Open:

```text
frontend/index.html
```

in your browser.
